# A Political Bias Classifier: NLP Analysis of Web-Scraped News Articles

## Overview
This repository is a project by Class Group 1 that contains a complete Natural Language Processing and machine learning pipeline designed to analyze and classify political bias in text data. The project covers the full lifecycle from data cleaning and Named Entity Recognition (NER) preprocessing to model training, scaling, and evaluation.

## Repository
* **`model_and_evaluation.ipynb`**: The primary Jupyter Notebook detailing the model training process, hyperparameter configurations, and evaluation metrics (accuracy, precision, recall, F1-score) for the trained classifiers.
* **`dataset_cleaner`**: The data cleaner
* **`Political_Bias_Cleaned_Balanced.csv`**: The fully preprocessed and class-balanced dataset used to train the machine learning models.
